# SBR Lift Pro Builder

A React + Vite + TypeScript project using shadcn-ui and Tailwind CSS.

## Getting Started

Prerequisites:
- Node.js and npm installed

Install dependencies:
```sh
npm install
```

Start the development server:
```sh
npm run dev
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
